## Acciones Inmediatas

### 1. Configurar Google Analytics
1. Ve a [analytics.google.com](https://analytics.google.com)
2. Crea propiedad
3. Copia el ID de medida
4. Reemplaza `GA_MEASUREMENT_ID` en `src/layouts/Layout.astro`
5. Descomenta el código

### 2. Configurar Google Search Console
1. Ve a [search.google.com/search-console](https://search.google.com/search-console)
2. Añade propiedad como dominio
3. Verifica con registro DNS TXT
4. Espera 24-48h para indexación

### 3. Desplegar a Netlify (recomendado)

```bash
# Opción 1: Deploy con CLI
npm install -g netlify-cli
netlify deploy --prod
# o conectar con GitHub
```

```bash
# Opción 2: Arrastrar carpeta
# 1. Ve a netlify.com
# 2. Arrastra la carpeta dist/
```

### 4. Registrar en directorios

Una vez desplegado, registrar en:
- there'sanai.com
- futurepedia.io
- aitools.guide
- alternativeto.net
