# Prompts para imágenes de miniaturas - IA Tools Hub

## Estilo visual recomendado para toda la web

Usar siempre una estética coherente:

- Formato horizontal 16:9.
- Estilo moderno, tecnológico y profesional.
- Fondo oscuro azul marino, degradados azules y cian.
- Elementos de inteligencia artificial: nodos, circuitos, paneles digitales, iconos abstractos, pantallas flotantes.
- Sin texto dentro de la imagen, salvo que sea imprescindible.
- Sin logos reales de marcas si no hay permiso de uso.
- Imagen limpia, con espacio visual para que el título del artículo respire.
- Aspecto premium, editorial y orientado a tecnología.

## Prompt base reutilizable

```text
Crea una imagen horizontal 16:9 para la miniatura de un artículo de una web sobre herramientas de inteligencia artificial.

Tema del artículo: [TEMA DEL ARTÍCULO]

La imagen debe transmitir: [IDEA PRINCIPAL DEL ARTÍCULO]

Estilo visual:
- diseño moderno, tecnológico y profesional
- fondo azul marino oscuro con degradados azul/cian
- elementos abstractos de inteligencia artificial, nodos conectados, circuitos, pantallas flotantes y luz digital
- composición limpia y premium
- aspecto editorial para blog tecnológico
- sin texto visible en la imagen
- sin logotipos reales de marcas
- alta calidad, iluminación cuidada, profundidad, contraste suave
- formato 16:9

Objetivo:
La imagen debe atraer clics, parecer profesional y estar relacionada con el contenido del artículo sin verse genérica.
```

# Prompt negativo recomendado

```text
Evita texto mal escrito, letras deformadas, logotipos reales, marcas registradas, caras extrañas, manos deformes, exceso de elementos, estilo infantil, colores chillones, baja resolución, aspecto genérico de banco de imágenes, composición desordenada o demasiado cargada.
```
